Have you ever wanted a status page that is highly customizable to match your brand and the way you want components to function; wait no longer, introducing Inticate Status. Customize everything from the ground-up from toggling if status boxes should show the specified URL or not, or even toggling if the status page should return each boxes current ping response from the server the status page is being hosted on.

## Features
- Easily configure all aspects of the application via a JSON file.
- Ability to configure each status box's properties individually.
- Configure status pages global settings such as support URL's, page title, and change content amongst the page.
